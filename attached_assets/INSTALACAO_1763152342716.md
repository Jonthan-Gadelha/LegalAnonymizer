# 📦 Guia de Instalação - Anonimizador de Processos

Este guia detalha como instalar e configurar a extensão do Chrome para anonimização de documentos jurídicos com suporte a PDF e OCR.

## 📋 Pré-requisitos

### Sistema
- **Google Chrome** versão 88 ou superior
- **Sistema Operacional**: Windows 10+, macOS 10.14+, ou Linux Ubuntu 18.04+
- **RAM**: Mínimo 2GB, recomendado 4GB+ para OCR
- **Espaço em disco**: ~10MB livres

### Verificação de Compatibilidade
1. Abra o Chrome e digite `chrome://version/` na barra de endereços
2. Verifique se a versão é 88 ou superior
3. Confirme que JavaScript está habilitado

## 🚀 Instalação Passo a Passo

### Passo 1: Download dos Arquivos
1. **Baixe** o arquivo `anonimizador-extensao.zip`
2. **Extraia** todos os arquivos em uma pasta permanente no seu computador
   - ⚠️ **Importante**: Não delete esta pasta após a instalação
   - Sugestão: `C:\Extensoes\Anonimizador\` (Windows) ou `~/Extensoes/Anonimizador/` (Mac/Linux)

### Passo 2: Ativar Modo Desenvolvedor
1. **Abra** o Google Chrome
2. **Digite** `chrome://extensions/` na barra de endereços
3. **Ative** a opção "Modo do desenvolvedor" no canto superior direito
   - O botão deve ficar azul quando ativado

### Passo 3: Carregar a Extensão
1. **Clique** em "Carregar sem compactação"
2. **Navegue** até a pasta onde extraiu os arquivos
3. **Selecione** a pasta `anonimizador-extensao`
4. **Clique** em "Selecionar pasta" (Windows) ou "Abrir" (Mac)

### Passo 4: Verificar Instalação
1. A extensão deve aparecer na lista com o nome "Anonimizador de Processos"
2. **Verifique** se está habilitada (botão azul)
3. **Procure** o ícone 🔒 na barra de ferramentas do Chrome

## 🔧 Configuração Inicial

### Teste Básico
1. **Clique** no ícone da extensão
2. **Cole** um texto de teste: "João Silva, CPF: 123.456.789-10"
3. **Clique** em "Anonimizar"
4. **Verifique** se o resultado mostra: "[NOME_REMOVIDO], CPF: [CPF_REMOVIDO]"

### Teste de PDF (Opcional)
1. **Abra** o arquivo `test-pdf.html` no navegador
2. **Execute** os testes disponíveis
3. **Verifique** se PDF.js e Tesseract.js carregaram corretamente

## 🛠️ Configurações Avançadas

### Permissões
A extensão solicita apenas permissões mínimas:
- **activeTab**: Para funcionar na aba atual (se necessário)
- **Nenhuma permissão de rede**: Garantia de privacidade total

### Otimização para OCR
Para melhor performance com OCR:
1. **Feche** abas desnecessárias antes de processar PDFs grandes
2. **Configure** o Chrome para usar mais memória:
   - Digite `chrome://flags/` na barra de endereços
   - Procure por "Memory"
   - Ative otimizações de memória se disponíveis

- [ ] Teste básico de anonimização funciona

## 🐛 Solução de Problemas

### Extensão não aparece
**Problema**: Ícone não aparece na barra de ferramentas
**Soluções**:
1. Verifique se está habilitada em `chrome://extensions/`
2. Clique no ícone de quebra-cabeça (extensões) e fixe a extensão
3. Reinicie o Chrome

### Erro ao carregar
**Problema**: "Falha ao carregar extensão"
**Soluções**:
1. Verifique se todos os arquivos foram extraídos corretamente
2. Confirme que o arquivo `manifest.json` está presente
3. Tente extrair novamente em uma pasta diferente

### PDF não funciona
**Problema**: PDFs não são processados
**Soluções**:
1. Verifique se o arquivo não está corrompido
2. Tente com um PDF menor (< 10MB)
3. Confirme que o PDF não está protegido por senha

### OCR muito lento
**Problema**: Processamento de OCR demora muito
**Soluções**:
1. Reduza o número de páginas (máximo 10 recomendado)
2. Feche outras abas e programas
3. Use PDFs com melhor qualidade de imagem

### Erro de memória
**Problema**: "Out of memory" durante OCR
**Soluções**:
1. Feche outras abas do Chrome
2. Reinicie o navegador
3. Processe PDFs menores ou com menos páginas

## 🔄 Atualização da Extensão

### Quando uma nova versão estiver disponível:
1. **Baixe** a nova versão
2. **Extraia** na mesma pasta (substitua os arquivos)
3. **Vá** para `chrome://extensions/`
4. **Clique** no botão de recarregar (🔄) da extensão
5. **Verifique** a nova versão no popup

## 🗑️ Desinstalação

### Para remover completamente:
1. **Vá** para `chrome://extensions/`
2. **Clique** em "Remover" na extensão
3. **Delete** a pasta dos arquivos extraídos
4. **Limpe** o cache do Chrome (opcional)

## 📞 Suporte

### Logs de Erro
Se encontrar problemas:
1. **Abra** as ferramentas de desenvolvedor (F12)
2. **Vá** para a aba "Console"
3. **Reproduza** o erro
4. **Copie** as mensagens de erro para análise

### Informações do Sistema
Para reportar problemas, inclua:
- Versão do Chrome
- Sistema operacional
- Tamanho do arquivo PDF (se aplicável)
- Mensagens de erro específicas

## 🔒 Segurança

### Verificação de Integridade
Para garantir que os arquivos não foram modificados:
1. **Verifique** se todos os arquivos listados estão presentes
2. **Confirme** que o `manifest.json` contém as permissões corretas
3. **Teste** a funcionalidade básica antes do uso em produção

### Boas Práticas
- **Mantenha** a pasta da extensão em local seguro
- **Não modifique** os arquivos da extensão
- **Teste** sempre com dados não sensíveis primeiro
- **Revise** os resultados antes de compartilhar documentos
